This blog is part of my personal academic homepage. I plan to use it for research notes, project updates, and occasional reflections on reinforcement learning, foundation agents, and systems work.

Because the site is built in a GitHub Pages-friendly structure, new posts can be added by creating Markdown files in the `_posts/` directory using the standard filename format:

```text
YYYY-MM-DD-title.md
```

That keeps writing lightweight while still making the homepage easy to maintain.
